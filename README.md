# Título: Loop Driver
 # Projeto-Final---Eduardo-Caio-e-Joao-Pedro
 # O jogo se consiste em uma corrida singleplayer de carro. O objetivo do jogo é percorrer um circuito gerado aleatoriamente e tentar percorrer a maior quantidade de metros. No jogo há 4 cenários diferente contendo obstáculos. Os obstáculo podem ser móveis ou não, e quando o jogador bate a corrida acaba. A cada volta completada no traçado da pista, a velocidade do carro é aumentada e os obstáculos mudam.
